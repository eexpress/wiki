# 本地化vala手册

```
▶ curl https://valadoc.org | grep devhelp | cut -c33- | sed 's/">.*$//'
| while read i; do wget -nc -c "http://www.valadoc.org"$i && 
tar -xf `basename $i` -C ~/.local/share/devhelp/books/; done
```
> wget的`-nc`参数，重复下载的文件，不自动添加后缀，直接不下载。因为下载经常会卡住。
